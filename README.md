# Task
 1. Выбрать из __первого файла__ 40 последних строк и записать во __второй файл__.
 2. Записать первые 10 строк из __второго файла__ в __третий файл__.
 3. Выбрать во __втором файле__ все строки которые содержат "коко", заменить строку "коко" на "куку" и дозаписать только первые три вхождения в __третий файл__.
 4. Оставить только уникальные строки в __третьем файле__ и получить количества каждой уникальной строки в виде: "КОЛ-ВО" "СТРОКА".
 
 # Solution
 1. ```bash 
    tail -n 40 first.txt > second.txt
    ```
 2. ```bash
    head -n 10 second.txt > third.txt
    ```
 3. ```bash
    sed 's/koko/kuku/g' second.txt | grep kuku | head -n 3 >> third.txt
    ```
 4. ```bash
    cat third.txt | uniq -c > third.txt
    ```
